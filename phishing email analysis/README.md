# Phishing Email Analysis

## Overview

This project documents the investigation of a phishing email impersonating Microsoft 365. The analysis focuses on identifying malicious indicators, examining email authentication, extracting Indicators of Compromise (IOCs), and mapping the attack to the MITRE ATT&CK framework.


## Objective

- Analyze a suspicious email.
- Verify sender authenticity.
- Examine email headers.
- Investigate embedded URLs.
- Identify Indicators of Compromise (IOCs).
- Assess the threat.
- Recommend mitigation strategies.


## Tools Used

- Gmail 
- MXToolbox Header Analyzer
- VirusTotal
- MITRE ATT&CK

## Final Verdict

**Classification:** Credential Phishing

**Severity:** High

The email impersonates Microsoft 365 and attempts to steal user credentials through a fake login page.
